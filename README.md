# Governance Token
Данный смарт-контракт распределяет входящие ETH на несколько адресов в долях, пропорциональных балансам governance токена.

# Компиляция, деплой и тестирование контракта
## Компиляция
Чтобы скомпилировать контракт, выполните:
```
brownie compile
```
## Деплой
Чтобы задеплоить проект, выполните:
```
brownie run deploy.py
```
## Тестирования
Для тестирования контракта в тестовой среде, выполните:
```
brownie console
```
 Для запуска юнит-тестирования, выполните:
 ```
  brownie test
 ```
